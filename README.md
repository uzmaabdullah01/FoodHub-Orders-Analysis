# FOOD HUB ORDER ANALYSIS (PYTHON)
Overview
This project analyzes FoodHub’s food delivery orders using Python. It includes data cleaning, exploratory data analysis (EDA), and visualizations to uncover patterns in cuisine popularity, order value trends, customer ratings, and preparation/delivery time performance.

## Dataset Features
- order_id  
- customer_id  
- cuisine_type  
- cost_of_the_order  
- day_of_the_week  
- rating  
- food_preparation_time  
- delivery_time  

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Most popular cuisine types  
- Order value distribution and high-value customers  
- Peak ordering times (day + time)  
- Delivery time vs rating relationship  
- Revenue-driving customer segments  

## Business Recommendations
- Improve staffing during peak demand  
- Optimize high-preparation-time cuisines  
- Promote high-revenue categories  
- Improve delivery speed to boost customer ratings  

## Project Files
- `foodhub_analysis.ipynb` – full analysis  
- `FoodHub_Report.pdf` – summary (optional)  
- `/images` – charts and visualizations
